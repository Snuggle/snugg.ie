Hypixel Bot

An unofficial Hypixel bot for Discord that shows you statistics and more.

View player's info

Using hypixel-player username you can view people's playercard. If their Discord account is linked, you can @handle#1234 them too.

And more!

By doing hypixel-help, you'll get messaged by Hypixel bot showing all of the available commands. More coming soon!

How do I add this bot?

To add this Discord bot to your server, click the "Add me!" button below. Select which server and click Authorise.

Once the bot is added to your server, type hypixel-help in a channel.
